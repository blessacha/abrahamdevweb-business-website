# AbrahamDevWeb Business Website

## Description

AbrahamDevWeb is a business website developed to showcase web development, mobile application development, and digital marketing services.

The website provides information about the company, services offered, technologies used, and a contact form for potential clients to get in touch.

## Features

* Responsive business website
* Service information pages
* Contact form
* About page
* Mobile and web development showcase
* Digital marketing information
* Modern HTML, CSS, JavaScript, and PHP implementation

## Services Offered

* Website Development
* Mobile Application Development
* Digital Marketing
* SEO Services
* Website Maintenance
* Mobile App Maintenance

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* PHP

### Tools & Platforms

* Firebase
* MySQL
* Google Cloud
* WordPress
* Figma
* Canva

## Project Structure

* HTML pages
* CSS stylesheets
* JavaScript functionality
* PHP contact form handling
* Images and media assets

## How to Run

1. Download the project.
2. Place the files on a PHP-enabled web server.
3. Configure email settings if using the contact form.
4. Open the website in a browser.
